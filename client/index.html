Welcome to the Transformers API

your place for all your transformers data needs

Routes:

Autobot Routes

  • GET /api/autobots - Get all Autobots
  • GET /users - Retrieve User and their Associated Autobots Requires Authentication
  • POST /api/autobots - Add an Autobot to the database JSON{name: STRING, color: STRING}
  • PUT /api/autobot - Update an autobot in the database{autobot_id: ID, autobot: STRING, color: STRING}
  • DELETE /api/autobot - Remove an autobot in the database{autobot_id: ID} | Requires Authentication

User Routes:

  • POST /api/auth/register - Register a user account JSON {email: STRING, password: STRING (minimum 6 characters)}
  • POST /api/auth/login - Login to account JSON {email: STRING, password: STRING}
